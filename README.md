# Harverst Managment
This is a simple management system for the Harverst project.

## Installation

### Requirements

- Node 20.x
- Bun 1.x

### Installation

Backend:
```bash
cd back-end
bun install
```

Frontend:
```bash
cd front-end
bun install
```

## Usage

### Backend

```bash
bun dev
```

### Frontend

```bash
cd fron-tend
bun dev
```
